Authenticates users against other WordPress instances using HTTP Basic Authentication against the WordPress REST API.

== Description ==

Uses HTTP basic authentication to authenticate against another WordPress instance. Requires the Basic Authentication
plugin installed on the remote site for authenticating and this same plugin also installed on the remote site in
order to add the user's email address to the /users/me REST endpoint.


== Installation ==

1. Upload and activate the plugin
2. Set a remote site
3. Forget
